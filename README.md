project name: Movie-Project

This project shall give the possibility to do varius things with movie based data.
It will store all data to a local database based on SQLite.

To use this project in full you will need an own API-Key from the website: "http://www.omdbapi.com"
API-Key:If you have one the easiest way is to create an .env file.
Put the key in this file like: API-KEY=your-key.
If you don't have a key at all you will just get some kind of demo.
You will also need to download SQLite to use a database as storage.

To ensure the program to run you have to import/install the mentioned requirements of the requirements.txt.
